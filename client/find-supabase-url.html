Method 1: From Your Browser Address Bar

When you're logged into your Supabase dashboard, look at the URL in your browser:

https://supabase.com/dashboard/project/[YOUR-PROJECT-ID]

Your project URL will be:

https://[YOUR-PROJECT-ID].supabase.co

Method 2: From Database Connection Info

  1. Go to your Supabase Dashboard
  2. Click on Database in the left sidebar
  3. Look for Connection info or Connection string
  4. You should see something like: Host: db.[project-id].supabase.co
  5. Your project URL is: https://[project-id].supabase.co

Method 3: From API Settings

  1. Go to Settings ‚Üí API
  2. Look for a field called Project URL or URL
  3. It should be displayed near your anon key

üí° Quick Check

If you can see your anon key, the project URL is usually in the same section. Look for:

  • Project URL
  • URL
  • Host
  • Endpoint

What to Look For

Your Supabase project URL will look like one of these:

https://abcdefghijklmnop.supabase.co
https://my-project-name.supabase.co
https://metaverse-project.supabase.co

Once You Find It

Update your .env file with:

VITE_SUPABASE_URL=https://your-actual-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-actual-anon-key
